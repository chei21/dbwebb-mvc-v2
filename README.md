## Klona projektet

```
git clone https://github.com/chei21/dbwebb-mvc-v2
```

## Kör projektet

Stå i projektmappen och kör följande kommando:

```
php -S localhost:8888 -t public
```

Gå till http://localhost:8888 i den webbläsare
